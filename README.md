🚚 Shipping Company Data Analysis Project (Tableau)
📝 Overview
This project is a Shipping Company Data Analysis Dashboard built using Tableau.
It focuses on analyzing shipping operations, delivery performance, and logistics efficiency to improve overall supply chain performance.
🎯 Objectives
Analyze shipping orders and delivery performance
Track on-time vs delayed deliveries
Identify high-performing and low-performing routes
Understand customer and shipment trends
Improve logistics efficiency and decision-making
🛠️ Tools Used
Tableau
Excel (Data Cleaning & Preparation)
Data Visualization
Basic Data Analysis
📊 Dashboard Features
Interactive filters (Region, Shipping Mode, Date, Customer Segment)
KPIs (Total Shipments, On-time Delivery Rate, Delays)
Charts for delivery performance and shipping costs
Route and region analysis
Trend analysis over time
Clean and interactive dashboard design
🔍 Key Insights
Some shipping routes experience higher delays than others
Delivery performance varies by region and shipping mode
On-time delivery rate is a key performance indicator
Data helps improve logistics planning and cost efficiency
📁 Files Included
Tableau Dashboard (.twbx / published link)
Dataset (Excel file)
README file
🚀 Conclusion
This project demonstrates how Tableau can be used in shipping and logistics analytics to optimize delivery performance, reduce delays, and improve supply chain efficiency through data-driven insights.
